# Lux_AI_S2


## Installation

```shell
git clone git@github.com:Ferdinand50/Lux_AI_S2.git
cd Lux_AI_S2
git clone https://github.com/Lux-AI-Challenge/Lux-Design-S2.git
```

## Run model
```shell
luxai-s2 main.py Bots/V0/main_V0.py -v 2 -o replay.json

```

## Create Submission
```shell
tar -czvf submission.tar.gz *

```
